Dynamic Example

In many cases we will need to pass Thymeleaf some data and ask it to generate HTML using that data.

In this example I added two items to the Thymeleaf model attribute: firstName and lastName.

I will need to use some special thymeleaf syntax to ask Thymeleaf to build some HTML around this data.

I will have to use some thymeleaf tags and then tell it which data I want to access.

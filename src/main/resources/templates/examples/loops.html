Again, being able to build HTML around a collection would be beneficial.

Again, HTML doesn't support loops, but Thymeleaf does.

Assume we have an ArrayList of student names and we want to add them to an unordered list. To build this HTML by hand, would be tedious. And we would have to update it every time the student list changed.

Instead we can pass the ArrayList to our Thymeleaf template with the isntructions to create some HTML for each student in the studentlist.

You can add or remove students from the studentNames ArrayList in our controller and Thymeleaf will always build the proper HTML for you!

Just like with Variables and injecting data into HTML. It is sometimes nice to build conditional logic into our HTML files.

Unfortunately, HTML is not a programming language and doesn't support conditional statements.

However, the Thymeleaf engine supports conditional statements and can conditional build, or not build HTML based on some boolean expression. Let's take a look.

For starters a boolean value was added to the model attributes named "shouldBuild". If it is true it will build the next element. An H1 that says Thymeleaf conditionally built me.

Thymeleaf conditionally built me, shouldBuild was true

Thymeleaf condtionally built me, but shouldBuild was false

You can tell thymeleaf to build something conditionally with the th:if= syntax. You can embed this syntax in any HTML element. And you can wrap it around multiple HTML elements by nesting all of them in a div element that has the th:if= conditional in it.
